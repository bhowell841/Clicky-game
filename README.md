# Clicky-game

## Overview
Memory game with React. This assignment require you to break up your application's UI into components, manage component state, and respond to user events.

## Technologies Used: 
<ul>
  <li>React</li>
  <li>ES6</li>
  <li>Bootstrap</li>
  <li>CSS</li>
</ul>
